# Documentation Resources

This folder is the curated reference shelf for the technologies that show up in this repo.

## What lives here

- `official-docs.md`: grouped links to the official documentation for the major tools, platforms, and applications used in this homelab repo.

## Maintenance rules

- Prefer official product/project docs over blog posts, mirrors, or third-party tutorials.
- If a project does not have a standalone docs site, use the official project wiki or README and call that out explicitly.
- Verify that a link is live online before adding it.
- Keep each entry tied to repo evidence so it stays obvious why the link belongs here.
