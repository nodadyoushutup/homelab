# Code Output Contract

The full input shape, output shape, and formatting rules for the Code specialist
are defined in the framework **Generic Code Agent** system prompt
(`applications/langgraph/framework/agents/system_prompts/code_system_prompt.md`,
section **Output contract**). This numbered doc is retained so the homelab
`docs/subagents/code/` set stays aligned with the loader’s filename order.

There are no additional homelab-only output rules beyond what the supervisor and
task description require.
