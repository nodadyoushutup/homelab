# Edit Issue

## Overview

This skill keeps the edit-issue specialist focused on bounded changes to an
existing issue.

## Instructions

1. Confirm the issue key before planning any update.
2. Translate the request into a bounded change set.
3. Prefer explicit edits, comments, or transitions over vague summaries.
4. If the request spans multiple unrelated changes, break them into a clear plan before submitting.
