# Create Issue

## Overview

This skill keeps the create-issue specialist focused on opening new work with a
complete payload.

## Instructions

1. Confirm that the request is for a new issue, not an update.
2. Collect the minimum creation fields: project, issue type, summary, and description.
3. Normalize ambiguous details into explicit draft fields before submitting.
4. Flag missing fields instead of guessing them silently.
