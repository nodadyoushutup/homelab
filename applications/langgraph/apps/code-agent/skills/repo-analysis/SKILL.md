# Repo Analysis

## Overview

This skill keeps the Code agent focused on file-backed evidence,
affected scope, and execution paths.

## Instructions

1. Start with the most relevant repo docs or config files before broad search.
2. Trace the narrowest path that answers the question.
3. Separate confirmed behavior from assumptions.
4. Return concise findings that another agent can act on directly.
